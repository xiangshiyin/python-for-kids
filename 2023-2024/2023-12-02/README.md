## Concepts
* Define a `function`
  ```python
  def function_name():
      <logic>
  ```
  or
  ```python
  def function_name():
      <logic>
      return <output>
  ```
  or
  ```python
  def function_name(param1, param2, ...):
      <logic>
  ```
  or
  ```python
  def function_name(param1, param2, ...):
      <logic>
      return <output>
  ```
* Call a function
  ```python
  function_name()
  ```
  or
  ```python
  function_name(param1=value1, param2=value2)
  ```
* Local and Global variables [[reference](https://www.geeksforgeeks.org/global-local-variables-python/)]
